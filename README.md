## 🖥️ What dots.mocr does

dots.mocr is a desktop app for reading text from many kinds of files. It is built for people who want to pull text from documents without a long setup.

You can use it to:

- Read text from scanned pages
- Pull text from screenshots
- Work with PDFs and image files
- Handle mixed content like forms, tables, and notes
- Copy text into other apps for editing or sharing

The app is designed to help you turn documents into text you can use right away.

## 🧼 Best results tips

Use these tips to get better text output:

- Scan at 300 DPI if you can
- Keep pages flat and straight
- Use good lighting for photos
- Crop out extra background
- Use clear, dark text on a light background
- Check for blur before you process the file

If a page has several columns, complex tables, or handwriting, you may need to review the output after the scan.

## 🔧 Common issues

### The app does not open
- Check that you downloaded the Windows file from the release page
- If the file came in a `.zip`, make sure you extracted it first
- Try running the app as administrator

### Windows blocks the file
- Open the file prompt
- Choose More info
- Select Run anyway if you trust the release source

### The text looks wrong
- Use a clearer image
- Scan again with higher quality
- Make sure the page is not tilted
- Remove shadows and glare

### The app feels slow
- Close other large apps
- Process one file at a time
- Use smaller images when possible

## 📄 What to expect from OCR

OCR works best when the source file is clear and simple. It can read many printed documents with good accuracy. It may struggle with:

- Handwriting
- Heavy blur
- Low light photos
- Decorative fonts
- Pages with strong shadows
- Broken or cut-off text

For best results, use files with sharp text and strong contrast
